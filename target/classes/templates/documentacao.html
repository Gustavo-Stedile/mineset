Documentação da Classe StrategyBase

Visão Geral

A classe StrategyBase é uma classe abstrata usada como base para estratégias customizadas.

Ela implementa Serializable e possui um controlador genérico para interação com o jogo.

Métodos

public void setController(Object ctrl): Define o controlador da estratégia.

public String toString(): Retorna uma string representando a estratégia.

Observações

A última atualização desta classe foi em 27/03/2025.

A classe pode ser estendida para criar estratégias personalizadas.
